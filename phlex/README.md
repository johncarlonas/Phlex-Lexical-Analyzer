# Phlex Frontend

This is the React-based user interface for the Phlex Lexical Analyzer.

## Tech Stack
- React 18
- Vite
- Vanilla CSS (Strict sharp-edge design)

## Features
- Real-time lexical workspace
- State persistence during navigation
- Structured token display with type-based styling

For full project documentation and setup instructions, please see the [Main README](../README.md).
